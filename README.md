## Week2-2 Git
* _Git_ 필수 명령어
  1. add : 커밋할 목록(Index 영역)에 추가
  2. commit : 커밋 만들기
  3. push : 내가 commit한 내용들을 Github에 추가하기

* _Git_ 상태 확인 명령어
  1. git show
  2. git log
  3. git shortlog
  4. git diff
  5. git status

## Week2-3 pull request

**_pull-request_**

_Pull Request_는 내가 수정한 코드를 내 GitHub 저장소의 브랜치에 올린 뒤, 원본 저장소 **소유자**에게 그 변경 내용을 병합해 달라고 요청하는 과정
